# Reflect — Rapid Prototyping Assignment

## Product
**Reflect** is a fitness and rehab tracking app (https://reflectsalus.app/) that automates post-practice and post-competition check-ins via SMS/WhatsApp. It delivers real-time athlete wellness data to coaches through dashboards, flag alerts, injury heatmaps, and an AI coaching assistant.

## Assignment
MPCS 51238 — Design, Build, Ship (Spring 2026, UChicago). Build 25 landing page iterations exploring radically different design directions, plus a gallery page. Pure HTML + CSS only.

## Tech Stack
- Pure HTML + CSS (no JavaScript, no frameworks)
- CSS-only visuals (gradients, geometric shapes, patterns — no external images)
- Each file is self-contained with inline `<style>` tags
- Deployed via GitHub + Vercel

## File Structure
- `index.html` — Gallery page with thumbnails linking to all 25 versions
- `v1.html` through `v25.html` — Individual landing page iterations
